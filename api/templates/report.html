## About Amicus Amicus is a tool that analyzes pull requests and provides feedback on the quality of the code. It is designed to be used by developers to improve the quality of their code and to help them make informed decisions about whether to merge a pull request. Amicus analyzes the code in a pull request and provides a score and a summary of the findings. The score is a number between 0 and 5, where 5 is the best possible score. The summary provides a high-level overview of the findings and recommendations for improving the code. Amicus also provides a detailed analysis of each file in the pull request. For each file, Amicus provides a summary of the code, a summary of the findings, and recommendations for improving the code.
